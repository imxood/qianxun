## 边界 (本次**不**做的事)

### 不重做 v2 基础设施

v2 已有 HookEvent 4 时机 / 双轴模式 / SubAgent 进程内 fork, 本工作项只做**叠加**, 不重写。

### 不做借鉴源完整移植

每个借鉴源都有自己完整体系, 千寻只取**所需片段**:
- octos: 只取 3-layer failover + 5 queue modes + hook exit-code, 不取 BLOCKED_ENV_VARS / 1500 行限制
- openclaw-mini: 只取双层循环 + Memory Flush, 不取 12 个 Gateway 模式 / Heartbeat 双层
- mempal: 只取 5 状态 + Gate, 不取 108 spec governance / cowork bus
- oh-my-opencode: 只取 hashline / 5-class background / 5-tier hook, 不取 11 agents / 62 hooks
- opencrust: 只取 skill lifecycle, 不取 WASM 插件 / self-update
- agent-spec: 只取 Verdict 4 态, 不取 BDD DSL / multi-VCS
- Karpathy: 只取 Simplicity First + Goal-Driven, 不取 4 准则

### 不做优先级变更

按 [设计/00_总览.md](../../设计/00_总览.md) §3 排好的顺序实施, 不擅自调整:
- P0 缺口 1 → 2 → 3 → 5 (缺口 4 后置)
- P1 缺口 9 → 6 → 12 → 10 → 14 → 11 → 7 → 13 → 8

### 不引入新外部依赖

14 缺口都用 Rust 标准库 + 现有 crate (tokio / async-trait / serde_json / time) 实现, 不引新 SDK。
